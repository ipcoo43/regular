- 필요한 것 : 로그가 찍힌 시간, 남은 힙, 네이트브 메모리의 양이다.

- ^L.+\[(\d\d:\d\d:\d\d).+\[(\d+)/(\d+).+\[(\d+)/(\d+).*$
- ((?:\d{2})(?:\:\d{2}){2}).*?\[\s(\d+).*?\[\s(\d+)
	- ( : 첫 번째 그룹을 캡쳐
	- (?:\d{2}) : 숫자 두 개가 연속적으로 나오고
	- (?: : 캡쳐안되는 그룹으로 묶은,
	- \:\d{2} : :45 같은 :뒤에 숫자 두 개가 오는 패턴이
	- ){2} : 두개 더 붙어서 12:34:56 패턴을 완성
	- ) : 이게 첫번째 그룹

	- .*?\[\s : "[ "을 만날 때까지 앞으로 나가서
	- (\d+) : 연속된 숫자만 취한다. 이게 두 번째 그룹
	- .*?\[\s(\d+) : 같은 패턴을 한 번 더 써서 세 번째 그룹
